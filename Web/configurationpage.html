🎵 SpotifyLocalSync

Synchronizuje publiczne playlisty Spotify z lokalną biblioteką muzyczną Jellyfin przy użyciu algorytmu dopasowania rozmytego (fuzzy matching).

Poświadczenia Spotify API

Utwórz aplikację na developer.spotify.com/dashboard i skopiuj Client ID oraz Client Secret. Wtyczka używa przepływu Client Credentials – nie wymaga logowania użytkownika.

Playlisty Spotify

Wklej adresy URL publicznych playlist Spotify – po jednym w każdej linii.
Akceptowane formaty: https://open.spotify.com/playlist/<id>, spotify:playlist:<id> lub samo ID playlisty.

Opcje dopasowania

Domyślnie 75. Wyższe wartości = mniej fałszywych dopasowań, ale też mniej znalezionych utworów.
Gdy włączone, zarówno tytuł jak i artysta muszą przekroczyć minimalny wynik. Wyłącz dla składanek lub kiedy metadane artysty są niekompletne.

Opcje playlisty

Dodawany przed nazwą pobraną ze Spotify, np. [Spotify] Top Hits Poland.
GUID użytkownika Jellyfin, który będzie właścicielem tworzonych playlist. Jeśli puste, automatycznie wybierany jest pierwszy administrator.
Gdy włączone, istniejąca playlista o tej samej nazwie zostanie zaktualizowana. Gdy wyłączone, za każdym razem tworzony jest nowy obiekt.

Harmonogram automatycznej synchronizacji

Ustaw 0 aby wyłączyć automatyczną synchronizację. Zadanie można zawsze uruchomić ręcznie w Pulpit → Zaplanowane zadania.
